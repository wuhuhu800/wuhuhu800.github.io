📋 功能1: 代码块一键复制

每个代码块右上角自动显示"复制"按钮，点击后变为"已复制"状态，2秒后自动恢复。

JavaScript 
function hello() {
    console.log('Hello, Cyberpunk!');
}

🔍 功能2: 图片灯箱效果

点击文章内的图片可以放大查看，支持 ESC 键关闭。图片带有 zoom-in 光标提示。

点击图片试试（示例图片）:

⬅️ ➡️ 功能3: 文章导航

文章底部显示上一篇和下一篇导航，快速切换相关文章。

← 上一篇 Python数据分析入门指南 下一篇 → 深度学习模型优化技巧总结

📊 功能4: 实时字数统计

文章页面左下角悬浮显示字数、代码行数、预计阅读时间。

🎉 功能6: 阅读完成庆祝

当滚动到文章95%位置时，会触发彩带动画和"阅读完成"提示。

🖼️ 功能7: 图片懒加载

页面外的图片使用懒加载，提升页面性能。图片加载前显示 shimmer 动画占位。
