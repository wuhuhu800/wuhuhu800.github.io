## ⚙️ 自定义配置

### 修改主题颜色
编辑 `assets/css/cyberpunk-theme.css` 中的 CSS 变量：
```css
:root {
  --accent-cyan: #00f5ff;    /* 主色调 - 青色 */
  --accent-purple: #bf00ff;  /* 副色调 - 紫色 */
  --accent-green: #00ff88;   /* 强调色 - 绿色 */
  --accent-orange: #ff8800;  /* 警告色 - 橙色 */
}
```

### 修改作者信息
编辑 `_config.yml`：
```yaml
author: 你的名字
description: 你的博客描述
email: 你的邮箱
github_username: 你的GitHub用户名
```

### 启用/禁用功能
编辑 `_config.yml` 中的 `theme_settings`：
```yaml
theme_settings:
  particle_background: true   # 粒子背景
  reading_progress: true      # 阅读进度
  toc_navigator: true         # 目录导航
  tts_enabled: true           # 语音朗读
  ai_assistant: true          # AI助手
```
